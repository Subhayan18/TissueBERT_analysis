# TissueBERT Analysis Pipeline

This repository contains all code used in the multi-phase analysis of the TissueBERT project.

## Project Phases

1. **Data Preparation** (`step_1_data_preparation`)
2. **Model Architecture** (`step_2_model_architecture`)
3. **Model Training** (`step_3_model_training`)
4. **Validation & Testing** (`step_4_validation_testing`)
5. **Clinical Application** (`step_5_clinical_application`)

Each phase is modular and documented for reproducibility.

## Usage

Clone the repo and follow instructions in each phase's README to run the analysis.
